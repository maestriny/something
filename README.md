# something (WIP)

A todo app built with React Native and Expo as part of the _Developing Mobile Apps with React Native_ capstone project on Coursera.

## Features

- Create and manage todos
- Mark todos as done and view completed items
- Multi-language support (EN, IT, DE, ES, FR)

## Tech Stack

- React Native / Expo (Expo Router)
- TypeScript
- Supabase
- Zustand + AsyncStorage
- React Native Reanimated
- React Hook Form + Zod
- i18next
